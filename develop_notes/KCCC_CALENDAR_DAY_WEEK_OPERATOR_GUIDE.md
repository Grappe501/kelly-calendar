# Day / Week operator guide (CC-08)

## Open the workspace

- Day: Calendar → **Day**, or `/calendar?view=day`
- Week: Calendar → **Week**, or `/calendar?view=week`
- Agenda remains available as a list fallback from the toolbar

## Scan the grid

- All-day Events appear in the top lane
- Timed Events are blocks on the hour grid
- Side-by-side blocks mean **visual overlap**, not automatically an unresolved conflict
- Conflict badges reflect **saved CC-06 conflict records**
- Events outside the visible hour window are called out — expand hours via layout query keys or use Agenda

## Navigate

- Previous / Next and Jump to today/this week keep search filters and layout preferences
- **Now** (on campaign-local today) scrolls to the current-time line
- Narrow screens stack week columns; use Day view or Agenda when space is tight

## Inspect an Event

- Click or activate an Event block to open the inspector (workspace stays open)
- Escape or Close dismisses the inspector
- **Open full event sheet** for authorized edit/reschedule (no drag/resize on the grid)

## Create an Event

1. Click an empty hour slot, use keyboard on a slot, or **Add Event**
2. Quick-add opens with a normalized campaign-local start
3. Confirm end time / duration and save only when ready
4. Selecting a slot alone does **not** create an Event

## Filters and saved views

Search chrome and saved views from CC-07 apply to Day and Week. Bookmarking the URL restores authorized filters and compatible layout preferences.
